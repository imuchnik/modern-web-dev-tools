Writing Less gulp task

  • Rename main.css to main.less
  • Install gulp less plugin
  • Install 'npm install --save-dev gulp-less'
  • Require it in your gulp file
  • Pipe it throught less preprocessor
  • Watch .less files rather then .css
  • Your file should look like this 
                    
             var less = require('gulp-less');
                    ...

              gulp.task('styles', function() {
                  gulp.src(['src/styles/main.less'])
                    .pipe(sourcemaps.init())
                    .pipe(less()) //<-- less pre prepocessor
                    .pipe(minifyCss())
                    .pipe(sourcemaps.write())
                    .pipe(gulp.dest('dist/styles'))
                    .pipe(browserSync.stream());
            });
                    ..

        //in default task
        gulp.watch('src/styles/**/*.less', ['styles']);

Lets similarly make our main.css modular

  • Crete a new directory call modules inside styles directory
  • Create 2 files called base.less and helpers.less inside our new directory
  • Copy and paste relevant bits into those 2 files
  • Import them into our main.less file

        @import "../../node_modules/normalize.css/normalize.css";
        @import "modules/base.less";
        @import "modules/helpers.less";
