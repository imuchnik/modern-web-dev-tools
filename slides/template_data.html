Templating engines like handlebars allow us to template our data

let's try it!

Download menu.json file here and place it into your project root directory

let's take a look at our json file.

Let add this data to our pages

  • Start by requiring it in your gulpfile 
    
    var menu = require('./menu.json');
  • Next add it to our data object 
        #in template task
     var data = {
        menu: menu.menuItems
  };
  • note how menuItems correspond to our json format
  • finally modify our menu.hbs to include it

     <p>Please vote for your favorite items</p>

    <ul class="section-list menu-list">
      {{#each menu}} //iterating through array of our items
       <li class="link" data-id={{id}}>
          <span class="menu-title">{{title}}</span>
            <span class="menu-count">{{count}}</span>
        </li>
      {{/each}}
    </ul>

Let's run gulp to see the result

You can now see how the same process can be applied to processing API calls results
