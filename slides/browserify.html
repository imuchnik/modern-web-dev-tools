Browserify

Browserify is a tool for CommonJs and npm modules bundling

Using browserify will enable us to create our own commonJs modules as well as use npm seamlessly

Setting up browserify gulp task

  • We will need to install browserify, vynil-source-stream and viny-buffer
  • Make sure to use --save-dev option
  • the last 2 packages help browserify work with gulp sreams
var browserify = require('browserify');
var source = require('vinyl-source-stream');
var buffer = require('vinyl-buffer');

gulp.task('scripts', function(){
  var b = browserify({
    entries: 'src/scripts/main.js',
    debug: true
  });

  b.bundle()
    .pipe(source('main.js'))
    .pipe(buffer())
    .pipe(sourcemaps.init({loadMaps: true}))
    .pipe(uglify())
    .pipe(sourcemaps.write('./'))
    .pipe(gulp.dest('dist/scripts/'))
    .pipe(browserSync.stream());
});
