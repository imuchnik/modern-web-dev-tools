Less variables

Less provides a variable syntax via @ symbol

For example

    

       
@yellow: #DDC83A;

.header-container {
   background: @yellow;
}

Let's practice using variables

  • Let's create another file called color_variables.less
  • Add the following

            @red: #A05035;
            @brown: #523E25;
            @light-bg: #F2F2EC;
            @dark-bg: #E4E5D8;

Mixins

Mixins look just like .less files, but less allows us to import them and use them as functions

let's try it.

  • Create mixins.less file
  • Add the following code 
    
.reset {
  margin: 0;
  padding: 0;
}

.rounded-corner(@radius) {
  border-radius: @radius;
}

Cont'd

  • Import the file into main.less, right after the variables import
  • Lets try to use it in other files
  • In base.less 
               
    
.section-heading {
  .reset;
}
.btn {
  display: block;
  max-width: 300px;
  margin: 0 auto;
  text-align: center;
  font-size: 1.6em;
  padding: 1em;
  .rounded-corner(4px);
}
