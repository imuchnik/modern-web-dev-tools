Unit testing

  • Allows us to test our js functions
  • Helps insure that changes don't break existing code
  • Promotes better code hygiene and organization

Mocha

  • Install mocha 
                 $ npm install --save-dev gulp-mocha
  • Write our gulp task 
                gulp.task('test', function(){
  gulp.src('test/*.js')
    .pipe(mocha())
});
